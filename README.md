# House Price Prediction Using Machine Learning

## Project Overview

This project aims to predict house prices using machine learning techniques. Various factors such as area, number of bedrooms, bathrooms, parking facilities, furnishing status, and other housing features are used to estimate the selling price of a house.

## Objectives

* Analyze housing market data
* Perform data preprocessing and visualization
* Train a machine learning model
* Predict house prices based on input features
* Evaluate model performance

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Dataset Features

* Area
* Bedrooms
* Bathrooms
* Stories
* Parking
* Main Road Access
* Guest Room
* Basement
* Air Conditioning
* Furnishing Status

## Machine Learning Algorithm

* Random Forest Regressor

## Results

The model successfully predicts house prices based on housing characteristics and achieves a good R² score on the test dataset.
